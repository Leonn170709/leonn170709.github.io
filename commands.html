Befehle

Befehle in Minecraft sind spezielle Anweisungen, die Spielern ermöglichen, das Spiel zu steuern und zu modifizieren.

Grundlegende Befehle

/help

Beschreibung: Zeigt eine Liste aller Befehle an.

/give

Beschreibung: Gibt einem Spieler ein bestimmtes Item.

Befehl: /give [Spieler] [Item] [Menge]

/tp

Beschreibung: Teleportiert einen Spieler zu einem anderen Spieler oder zu bestimmten Koordinaten.

Befehl: /tp [Ziel] [Zielort]

Erweiterte Befehle

/summon

Beschreibung: Beschwört eine Entität an einem bestimmten Ort.

Befehl: /summon [Entität] [Koordinaten]

/effect

Beschreibung: Verleiht einem Spieler einen Status-Effekt.

Befehl: /effect [Spieler] [Effekt] [Dauer] [Stärke]

/gamemode

Beschreibung: Ändert den Spielmodus eines Spielers.

Befehl: /gamemode [Modus] [Spieler]

/weather

Beschreibung: Ändert das Wetter im Spiel.

Befehl: /weather [Typ]

/time

Beschreibung: Ändert die Tageszeit im Spiel.

Befehl: /time set [Zeit]

Experten Befehle

/xp

Beschreibung: Gibt einem Spieler Erfahrungspunkte.

Befehl: /xp [Menge] [Spieler]

/kill

Beschreibung: Tötet eine Entität.

Befehl: /kill [Ziel]

/locate

Beschreibung: Findet die Koordinaten der nächsten Struktur eines bestimmten Typs.

Befehl: /locate [Struktur]

/setblock

Beschreibung: Platziert einen Block an den angegebenen Koordinaten.

Befehl: /setblock [Koordinaten] [Block]

/fill

Beschreibung: Füllt einen Bereich mit einem bestimmten Block.

Befehl: /fill [von Koordinaten] [bis Koordinaten] [Block]

/clone

Beschreibung: Kopiert Blöcke von einem Bereich in einen anderen.

Befehl: /clone [von Koordinaten] [bis Koordinaten] [Ziel Koordinaten]
