Minecraft Pi

Minecraft Pi ist eine spezielle Version von Minecraft, die für den Raspberry Pi entwickelt wurde.
Diese Version bietet grundlegende Funktionen und ist ideal für Programmieranfänger, um das Spiel zu modifizieren und zu erweitern.
Allerdings ist sie nicht so umfangreich wie andere Editionen.


Geschichte

Minecraft Pi Edition wurde von Mojang entwickelt und 2013 veröffentlicht.
Diese Version basiert auf der Pocket Edition und wurde speziell für den Raspberry Pi angepasst.
Sie bietet Spielern die Möglichkeit, mit Python zu programmieren und einfache Modifikationen vorzunehmen, wodurch sie ideal für Schulen ist.

Mehr über Minecraft Pi

Minecraft Pi bietet eine Vielzahl an Features:

  1. Unterstützung für Python-Programmierung
  2. Grundlegende Minecraft-Funktionen
  3. Ideal für Bildungszwecke
  4. Leichtgewichtige Version für den Raspberry Pi
Vor- und Nachteile

Pros

  • Leichtgewichtig und läuft gut auf dem Raspberry Pi
  • Unterstützt Python-Programmierung
  • Ideal für Bildungszwecke und Programmieranfänger

Cons

  • Begrenzte Funktionen im Vergleich zu anderen Editionen
  • Nur verfügbar auf dem Pi und Debian based Linux Distros die ARMv7 unterstützen
  • Nur eine Version verfügbar
